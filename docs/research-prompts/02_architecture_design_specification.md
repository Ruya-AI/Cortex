# Prompt 02: Architecture Design Specification

Act as a Chief Architect designing a production-grade QA Platform.

Using the research and planning document below, create a complete end-to-end architecture design.

Do not write implementation code.

Produce a detailed architecture specification.

Include:

# 1. High-Level Architecture
- Overall system architecture
- Major components
- Communication patterns
- Data flow
- External dependencies
- Architecture diagram description
- Component relationships
- Runtime workflow explanation

# 2. System Components
For every component define:
- Component Name, Purpose, Responsibilities, Inputs, Outputs, Dependencies, Failure scenarios, Scaling strategy

# 3. Agentic AI Architecture
- Agent roles, responsibilities, communication model, memory strategy, tool usage strategy
- Planning/execution loops, validation mechanisms, human approval checkpoints
- Per-agent: Objective, Capabilities, Tools, Knowledge sources, Decision boundaries, Interactions

# 4. Data Architecture
- Database design, data ownership, storage choices, event models, caching strategy, search/indexing, audit history

# 5. Security Architecture
- Authentication, authorization, tenant isolation, secrets management, data protection, compliance

# 6. Infrastructure Architecture
- Deployment model, cloud architecture, containers, Kubernetes, CI/CD, monitoring, logging, alerting

Research Input:
Use the research and planning document already produced.
