# Prompt 05: Implementation Roadmap

You are a VP of Engineering creating an implementation roadmap.

Based on the architecture documentation, create a phased engineering execution plan.

Do not write code.

Create:
- Phase 1: Foundation (infrastructure, core services, dev setup, initial architecture)
- Phase 2: Core Platform (features, components, dependencies, deliverables)
- Phase 3: AI Capabilities (agents, workflows, integrations)
- Phase 4: Production Readiness (security, performance, monitoring, reliability)

For each phase: Objectives, Tasks, Dependencies, Engineering risks, Acceptance criteria, Expected outputs.

Architecture:
Use and follow the architecture and design documents already created.
