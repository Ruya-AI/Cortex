# Prompt 04: Agentic Architecture Specification

Act as an AI Systems Architect specializing in autonomous agent platforms.

Design the agentic architecture for this QA Platform.

Create a complete AI agent specification.

Include:
- Agent System Overview (why agents, responsibilities, autonomous vs controlled)
- Per-agent: Name, Mission, Objective, Capabilities, Reasoning, Tools, Input, Output, Memory, Knowledge, Decision Process, Failure Handling, Human Escalation
- Agent orchestration model, communication protocol, task planning lifecycle, execution lifecycle, evaluation/verification, feedback loops, learning/improvement

Follow: explicit planning, tool boundaries, observable execution, deterministic validation, safe autonomy, human oversight.

Input:
Use Complete System Architecture Design document.
